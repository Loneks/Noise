# Noise

Це навчальний проєкт сайту "Інтерактивний музичний музей".

Сайт зроблений для того, щоб користувач міг переглядати інформацію про музику, жанри, виконавців, альбоми, новини і різні матеріали.

Це не музичний плеєр, а інформаційний сайт про музику.

На сайті є:

- головна сторінка
- таймлайн по десятиліттях
- жанри
- виконавці
- альбоми
- новини
- мультимедійні матеріали
- пошук
- фільтр альбомів
- адмін-панель

На головній сторінці можна вибирати десятиліття. Після цього змінюється текст, картинки і рейтинг альбомів та треків.

Проєкт зроблений на ASP.NET Core MVC.

Використовувались:

- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- HTML
- CSS
- JavaScript
- Bootstrap

База даних використовується SQLite.

Підключення до бази знаходиться у файлі appsettings.json:

"DefaultConnection": "Data Source=noise.db"

У базі даних зберігаються жанри, виконавці, альбоми, країни, новини і мультимедійні матеріали.

Для роботи з базою використовується Entity Framework Core.

Також в проєкті є адмін-панель. Через неї можна додавати, редагувати і видаляти дані на сайті.

Щоб запустити проєкт, треба виконати команди:

dotnet restore

dotnet build

dotnet run

Після запуску сайт відкривається за адресою:

http://localhost:5157

Основні папки проєкту:

Controllers - контролери

Models - моделі

Views - сторінки сайту

Data - робота з базою даних

wwwroot - стилі, картинки і JavaScript

Migrations - міграції бази даних

Проєкт зроблений для навчальної роботи.
